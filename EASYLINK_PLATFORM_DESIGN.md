# EasyLink 平台级控制台设计
## Platform Dashboard - 多商户管理架构

---

## 🎯 核心理念

> **"您不是 King-Chicken 的员工，您是 EasyLink 的 CEO"**

King-Chicken 只是您平台上的一个**商户（Merchant）**，就像 Shopify 的商家或 Stripe 的客户一样。

## 🎨 界面设计原则（参考 Stripe）

### 1. 色彩系统
```
主色: #FF6B00 (EasyLink品牌橙)
背景: #0A0A0F (深色专业感)
卡片: #12121A (略浅于背景)
成功: #10B981
警告: #F59E0B
错误: #EF4444
文字: #FFFFFF (主) / #9CA3AF (次)
```

### 2. 布局特点
- **左侧导航**：图标 + 文字，可收起
- **顶部栏**：搜索 + 通知 + 用户头像
- **内容区**：卡片式布局，清晰分区
- **数据可视化**：图表优先，数字突出

### 3. 交互设计
- 商户切换使用下拉菜单（像 Stripe 切换账户）
- 关键操作有确认弹窗
- 数据加载有骨架屏
- 实时数据自动刷新

## 🔐 权限体系

| 角色 | 权限 |
|------|------|
| **超级管理员** | 所有权限，包括创建管理员 |
| **运营经理** | 查看数据、管理商户、配置WhatsApp |
| **财务人员** | 查看结算、处理对账 |
| **技术支持** | 查看日志、配置API、查看交易详情 |
| **只读用户** | 仅查看数据，无操作权限 |
