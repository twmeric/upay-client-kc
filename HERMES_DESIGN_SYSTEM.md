## 🎯 設計原則

### 1. 奢華簡約
- 大量留白 (Negative Space)
- 精緻細節 (1px 優雅邊框)
- 高對比度文字

### 2. 質感層次
- 細微陰影: `0 2px 8px rgba(139, 90, 43, 0.08)`
- 卡片圓角: `8px` (優雅但現代)
- 漸層過渡: 橙色漸層 `linear-gradient(135deg, #FF6B00, #E55A00)`

### 3. 互動反饋
- Hover: 輕微上浮 + 陰影加深
- Press: 微縮放 (0.98)
- 過渡: `all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`

## 📦 組件規範

### 按鈕
```
主按鈕:
- 背景: 愛馬仕橙漸層
- 文字: 白色
- 圓角: 8px
- 高度: 48px
- Hover: 深橙漸層 + 陰影

次要按鈕:
- 背景: 透明
- 邊框: 1px 琥珀金
- 文字: 焦糖棕
- Hover: 琥珀金背景
```

### 卡片
```
- 背景: 米白 (#F5F0EB)
- 邊框: 1px 淺灰
- 圓角: 8px
- 陰影: 0 2px 8px rgba(0,0,0,0.04)
- Hover: 陰影加深
```

### 表格
```
- 表頭: 焦糖棕背景 + 白色文字
- 行: 象牙白/米白交替
- Hover: 琥珀金淺色背景
- 邊框: 1px 淺灰
```
